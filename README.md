# expo-push-easy docs

Landing page for the [`expo-push-easy`](https://www.npmjs.com/package/expo-push-easy) library — a push notification library that works with Expo Push and FCM v1 from any JS runtime (Node, Convex, Cloudflare Workers, Bun, Deno).

Built with [Vite](https://vitejs.dev/) + [React](https://react.dev/).

## Development

```bash
npm install
npm run dev
```

Runs at `http://localhost:5173`.

## Build

```bash
npm run build
```

Output goes to `dist/`.
